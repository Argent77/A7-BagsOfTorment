Обзор
~~~~~

Мод для игры Planescape Torment: Enhanced Edition.

Мод добавляет в игру несколько уникальных контейнеров, которые можно приобрести у некоторых торговцев, или добыть иным путем.
Один из контейнеров даже запускает небольшой квест.


Установка
~~~~~~~~~

Это мод на основе WeiDU, что означает, что его очень легко установить. Просто распакуйте скачанный архив в каталог с установленной
игрой и запустите файл "setup-BagsOfTorment.exe".

Note:
PST:EE patch version 3.1.4 and earlier contains a bug that will reset modded content whenever the party enters or resets the Modron Maze.
To work around this issue, it is strongly recommended to download and install Generalized Biffing (https://github.com/SpellholdStudios/generalized_biffing/releases/latest) as the final mod in the mod list.


Компоненты
~~~~~~~~~~

1. Контейнеры для PST:EE (основной компонент)

Добавляет в игру следующие контейнеры:
- Кожаная сумка бесов:          Можно приобрести у торговца на рыночной площади Хайва.
                                Хранит до 50 предметов.
- Складской кристалл модронов:  Может выпасть с Конструкта Злого Колдуна в Лабиринте модронов.
                                Подсказка: Быстрое убийство конструкта может обернуться чем-то необычным.
                                Хранит до 200 предметов.
- Карманная сингулярность:      Этот экзотический артефакт можно приобрести в Лавке Редкостей Вришики в районе Клерков.
                                Но вам еще предстоит выяснить, как заставить его работать...
                                Хранит до 500 предметов.


2. Увеличить вместимость контейнеров (требует установки основного компонента)

Это исправление увеличивает первоначальную вместимость складов до 32000
