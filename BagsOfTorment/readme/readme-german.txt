Übersicht
~~~~~~~~~

Eine Mod für Planescape Torment: Enhanced Edition.

Diese Mod installiert eine Reihe von Behältern, die in diversen Geschäften erworben werden können. 
Es gibt auch eine kleine Aufgabe für einen der Behälter zu lösen.


Installation
~~~~~~~~~~~~

Dies ist eine WeiDU Mod. Das bedeutet, sie ist sehr einfach zu installieren. Entpacke die 
heruntergeladene Zipdatei in das Spieleverzeichnis und starte "setup-BagsOfTorment.exe" (Windows).


Komponenten
~~~~~~~~~~~

1. Taschen für PST:EE (Hauptkomponente)

Installiert die folgenden Behälter:
- Teufelslederbeutel:      Kann von einem Händler auf dem Marktplatz im Stock erworben werden.
                           Kann bis zu 50 Gegenstände aufnehmen.
- Modron-Speicherkristall: Kann vom Bösen Zauberer-Konstrukt im Modron-Irrgang geraubt werden.
                           Tip: Es kann sich lohnen, das Konstrukt schnell zu besiegen.
                           Kann bis zu 200 Gegenstände aufnehmen.
- Taschensingularität:     Kann von Vrischikas Kuriositätenladen im Bezirk der Kuratoren erworben 
                           werden. Du musst zuerst herausfinden, wie man diesen Behälter nutzen 
                           kann. Kann bis zu 500 Gegenstände aufnehmen.


2. Kapazität der Taschen erhöhen (benötigt Hauptkomponente)

Diese Komponente erhöht die Kapazität der Behälter auf über 32000.
